Configuración del Grid Container:

  • Un contenedor #container que usa Grid Layout con 3 columnas y 5 filas.
  • Las filas tienen alturas variadas (10%, 15%, 50%, 10%, 15%).

Estilos para las Cajas:

  • Todas las cajas tienen un fondo rojo y texto blanco por defecto.
  • Se cambió el color de fondo de varias cajas específicas utilizando nth-child.

Distribución de las Cajas en la Cuadrícula:

  • La primera caja ocupa todas las columnas (grid-column: 1 / 4).
  • La segunda caja ocupa las dos primeras columnas (grid-column: 1 / 3).
