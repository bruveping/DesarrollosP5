Ejemplo de Display: Inline-Block

Descripción

El valor inline-block para la propiedad display combina las características de inline y block. Permite que un elemento ocupe solo el espacio necesario para su contenido, como inline, pero también permite definir un ancho y alto, como block. Esto es útil para elementos que necesitas alinear horizontalmente y controlar sus dimensiones.
