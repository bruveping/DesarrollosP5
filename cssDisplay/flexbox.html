Ejemplo de Flexbox

Descripción

El modelo de caja flexible (o flexbox) en CSS es un método de diseño que permite distribuir el espacio de manera eficiente entre los elementos de un contenedor flexible, incluso cuando su tamaño es desconocido o dinámico. Flexbox es ideal para aplicaciones de diseño de una sola dimensión.
