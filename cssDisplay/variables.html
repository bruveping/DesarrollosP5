Ejemplo de Variables en CSS

Descripción

Las variables en CSS, también conocidas como custom properties, permiten definir valores reutilizables que pueden aplicarse en múltiples partes de un documento CSS. Utilizando :root, puedes declarar variables globales y emplear la función var() para utilizarlas en tu hoja de estilo, facilitando la gestión y modificación de estilos.
