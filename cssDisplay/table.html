Ejemplo de Display: Table

Descripción

El valor table para la propiedad display convierte un contenedor en un contenedor de tabla y sus elementos hijos en elementos de tabla. Esto es útil para crear diseños similares a una tabla sin usar elementos <table>, <tr> o <td>. Permite una estructura más semántica y controlada de los elementos.
