Ejemplo de Diseño Responsivo

Descripción

El diseño responsivo en CSS se refiere a la capacidad de un sitio web para adaptarse a diferentes tamaños de pantalla y dispositivos. Utilizando media queries y propiedades como flex y grid, puedes crear diseños que se ajusten fluidamente, mejorando la experiencia del usuario en dispositivos móviles y de escritorio. En este caso, si se modifica el tamaño de la ventana, se modifica el diseño
