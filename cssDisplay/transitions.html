Ejemplo de Transiciones en CSS

Descripción

Las transiciones en CSS permiten cambiar gradualmente una propiedad CSS de un valor a otro durante un periodo de tiempo específico. Utilizando la propiedad transition, puedes definir qué propiedades serán animadas y la duración de la animación, creando así efectos visuales fluidos y atractivos. Coloca el mause sobre el cuadrado.
